# 🚀 ToolHub

A modern online file conversion toolkit built with **Python Flask**.

ToolHub allows users to convert, merge, compress and manage PDF, Image and Office files with a clean and responsive interface.

---

## ✨ Features

### 📄 PDF Tools
- ✅ PDF to Word
- ✅ Word to PDF
- ✅ Merge PDF
- ✅ Split PDF

### 🖼 Image Tools
- ✅ JPG to PDF
- ✅ Image Compressor
- ✅ Background Remover

### 🚀 Coming Soon
- OCR (Image → Text)
- Compress PDF
- Rotate PDF
- PDF to JPG
- Excel to PDF
- PowerPoint to PDF

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/mrayyan92/Toolhub-.git
```

Move into the project

```bash
cd Toolhub-
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

Windows

```bash
venv\Scripts\activate
```

Install requirements

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```
